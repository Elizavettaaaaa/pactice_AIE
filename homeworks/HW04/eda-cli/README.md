# EDA-CLI HTTP Service (HW04)

HTTP-сервис для оценки качества датасетов на базе FastAPI.
Является расширением проекта HW03.

## Установка и запуск

1. **Установка зависимостей:**
   ```bash
   uv sync
   ```
2. **Запуск сервера (Uvicorn):**
Для запуска сервиса выполните следующую команду:
   ```bash
   uvicorn eda_cli.api:app --reload --port 8000
   ```

## Эндпоинты

* **GET `/health`**: проверка работоспособности сервиса.
* **POST `/quality`**: оценка качества на основе переданных метрик (JSON).
* **POST `/quality-from-csv`**: загрузка файла и оценка качества.
* **POST `/quality-flags-from-csv`**: дополнительный эндпоинт.
  * Принимает CSV-файл.
  * Возвращает детальные флаги качества:
    * Проверка подозрительных дубликатов ID (`has_suspicious_id_duplicates`)
    * Подсчет константных столбцов (`constant_columns_count`)
